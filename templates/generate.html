Peptide Generator

Page dedicated to ACP sequence generation. Select a generative model and number of sequences to generate. Outputs generated sequences with cytotoxicity labels.

Important note: In the MASTER THESIS, rnn_mp model was used and evaluated.
Other models were added later and have not been thoroughly tested

Available generative models:

  • rnn_mp — main model used in the thesis, able to design experimentally validated ACPs
  • progen_mp — large transformer model fine-tuned on MPs
  • progen_acp — large transformer model fine-tuned on ACPs only
